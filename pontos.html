    • Points (pt)

    A unidade de medida point (ponto, abreviada como pt) é uma unidade tipográfica tradicionalmente definida como 1/72 de polegada. É utilizada principalmente na impressão e tipografia para medir o tamanho da fonte, espaçamento e outros elementos de texto, essa unidade é raramente usada para web design. Um ponto tem um valor de aproximadamente 0,3527 milímetros ou 0,0138 polegadas. Imprimível: É a medida padrão para design gráfico e tipos impressos, garantindo um tamanho consistente do tipo em diferentes contextos de impressão.

    Tipografia: É usada para definir o tamanho da fonte, o entrelinhamento (espaçamento entre linhas) e outras medidas de layout de texto em um documento. Diferença para pixels: Ao contrário de pixels (px), os pontos não são afetados pela resolução da tela, o que pode torná-los menos ideais para design na web.
